# Closed-test recruiting

Drafts and notes for getting the 12+ testers needed to satisfy Google's
14-day closed-testing requirement before Production access.

**Before posting:** get your opt-in link from Play Console -> Closed
testing -> Testers tab -> "Copy link" (looks like
`https://play.google.com/apps/testing/com.meetanor.app`). Replace
`[OPT-IN LINK]` below with it.

## Make recruiting actually work

1. **Use a Google Group for the tester list, not individual emails.**
   In Play Console -> Closed testing -> Testers, create/point to a
   Google Group (e.g. `anor-testers@googlegroups.com`) and set the group
   to allow anyone to join. Then strangers from a swap community can join
   the group themselves and opt in — no manual per-email adding.
   Individual-email lists don't scale for swaps.
2. **Reciprocity is the deal.** Tester-swap communities run on "I test
   yours, you test mine." You'll need to install the other devs' apps and
   leave them on a device (an emulator is fine) for their 14 days too.
   Budget a little time for it — it's the cost of the fast path.
