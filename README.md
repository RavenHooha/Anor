# Anor

Anor shows you who's actually nearby *right now* — the same coffee shop, bar, or block — so people can meet in real life instead of doomscrolling. Privacy-first: no ads, no data selling.

Built with Expo / React Native, Supabase (Postgres + PostGIS, RLS-enforced), and Bluetooth-LE proximity on Android with GPS elsewhere.

## The point of it

Anor is the funding mechanism, not the mission. The mission is the model it runs on:

- **Sufficiency, then give the rest.** After real operating costs and a sufficient founder living, **100% of net profit goes to housing** — the Newman's-Own model. Default recipient: Habitat for Humanity WNC.
- **Privacy as architecture, not policy.** What isn't stored can't leak, be sold, or be subpoenaed. The privacy commitments are enforced in the database and the code, not just promised in a document.
- **Replicable on purpose.** The model is meant to be copied, not hoarded.
